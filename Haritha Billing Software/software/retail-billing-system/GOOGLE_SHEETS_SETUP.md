## 1. Google Cloud Project Setup

### Step 1: Create a Google Cloud Project
1. Go to **https://console.cloud.google.com/**
2. Click the project dropdown at the top → **"New Project"**
3. Name it: `haritha-billing-system` → Click **Create**
4. Wait for the project to be created, then select it

### Step 2: Enable Google Sheets API
1. In the left sidebar → **APIs & Services** → **Library**
2. Search for **"Google Sheets API"**
3. Click on it → Click **"Enable"**
4. Also search and enable **"Google Drive API"** (required for sharing)

---

## 2. Service Account & Credentials

### Step 3: Create a Service Account
1. Go to **APIs & Services** → **Credentials**
2. Click **"+ Create Credentials"** → **"Service Account"**
3. Fill in:
   - **Service account name:** `haritha-billing-sa`
   - **Service account ID:** (auto-filled)
   - **Description:** `Service account for Haritha Billing Software`
4. Click **Create and Continue**
5. For Role, select: **"Editor"** (or "Basic → Editor")
6. Click **Continue** → **Done**

### Step 4: Generate JSON Key
1. Click on the service account you just created
2. Go to the **"Keys"** tab
3. Click **"Add Key"** → **"Create new key"**
4. Select **JSON** → Click **Create**
5. A JSON file will download automatically — **save it carefully!**

### Step 5: Place the Credentials File
Copy the downloaded JSON file to:
```
retail-billing-system/config/credentials.json
```

The file looks like this (your values will be different):
```json
{
  "type": "service_account",
  "project_id": "haritha-billing-system",
  "private_key_id": "abc12345...",
  "private_key": "-----BEGIN PRIVATE KEY-----\nMIIEvgIBADA...\n-----END PRIVATE KEY-----\n",
  "client_email": "haritha-billing-sa@haritha-billing-system.iam.gserviceaccount.com",
  "client_id": "1234567890",
  "auth_uri": "https://accounts.google.com/o/oauth2/auth",
  "token_uri": "https://oauth2.googleapis.com/token",
  "auth_provider_x509_cert_url": "https://www.googleapis.com/oauth2/v1/certs",
  "client_x509_cert_url": "https://..."
}
```

> **IMPORTANT:** Never commit credentials.json to version control. It is already listed in .htaccess to block public access.

## 3. Google Spreadsheet Setup

### Step 6: Create the Spreadsheet
1. Go to **https://sheets.google.com/**
2. Click **"+ Blank"** to create a new spreadsheet
3. Rename it to: **`Haritha Billing System DB`** (click the title at the top)

### Step 7: Get the Spreadsheet ID
The spreadsheet ID is in the URL:
```
https://docs.google.com/spreadsheets/d/  <<SPREADSHEET_ID_HERE>>  /edit
```
Copy that long string — you'll need it for `.env`

### Step 8: Share with Service Account
1. Click the **"Share"** button (top right of the spreadsheet)
2. In the email field, paste the **client_email** from your credentials.json
   - Example: `haritha-billing-sa@haritha-billing-system.iam.gserviceaccount.com`
3. Set permission to **"Editor"**
4. **Uncheck "Notify people"** (service accounts don't have inboxes)
5. Click **Share**

## 5. Apps Script (Optional Automation)

This Google Apps Script adds useful features like auto-formatting, data validation, and summary formulas.

### How to Open Apps Script:
1. Inside your Google Spreadsheet, go to **Extensions** → **Apps Script**
2. Delete any existing code
3. Paste the entire script below
4. Click **Save** (💾) → name the project `HarithaBillingAutomation`
5. Click **Run** → Select `setupSheets` → Authorize when prompted

---

### Complete Apps Script Code

```javascript
/**
 * Haritha Billing Software — Google Apps Script
 * Auto-setup headers, formatting, and data validation
 * Paste this in Extensions > Apps Script
 */

// ============================================================
// CONFIGURATION
// ============================================================
var SHEET_CONFIG = {
  Products: {
    headers: ['ID','Name','Category','Price','GST_Percent','Barcode','Quantity','Unit','Description','Created_At','Updated_At'],
    colWidths: [160,200,120,80,80,130,70,70,200,160,160]
  },
  Bills: {
    headers: ['Bill_No','Date','Customer_Name','Customer_Phone','Items_JSON','Subtotal','GST_Amount','Discount','Net_Total','Round_Off','Final_Amount','Payment_Mode','Bill_Type','Status','Created_At'],
    colWidths: [140,100,150,120,300,90,90,80,90,80,100,90,80,70,160]
  },
  GST_Bills: {
    headers: ['GST_Bill_No','Date','Customer_Name','Customer_GSTIN','Customer_Address','Customer_State','Items_JSON','Taxable_Amount','CGST_Amount','SGST_Amount','IGST_Amount','Total_GST','Discount','Net_Total','Round_Off','Final_Amount','Payment_Mode','Inter_State','Status','Created_At'],
    colWidths: [140,100,150,140,200,120,300,100,90,90,90,90,80,90,80,100,90,80,70,160]
  },
  Stock_Log: {
    headers: ['Log_ID','Product_ID','Product_Name','Change','Type','Note','Created_At'],
    colWidths: [160,160,180,70,100,200,160]
  }
};

// ============================================================
// MAIN SETUP — Run this first!
// ============================================================
function setupSheets() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  
  for (var sheetName in SHEET_CONFIG) {
    var config  = SHEET_CONFIG[sheetName];
    var sheet   = ss.getSheetByName(sheetName);
    
    // Create sheet if it doesn't exist
    if (!sheet) {
      sheet = ss.insertSheet(sheetName);
      Logger.log('Created sheet: ' + sheetName);
    }
    
    // Set headers in row 1
    var headerRange = sheet.getRange(1, 1, 1, config.headers.length);
    headerRange.setValues([config.headers]);
    
    // Style the header row
    headerRange.setBackground('#2563eb')
               .setFontColor('#ffffff')
               .setFontWeight('bold')
               .setFontSize(11);
    
    // Set column widths
    for (var c = 0; c < config.colWidths.length; c++) {
      sheet.setColumnWidth(c + 1, config.colWidths[c]);
    }
    
    // Freeze top row
    sheet.setFrozenRows(1);
    
    // Auto-resize row height
    sheet.setRowHeight(1, 32);
    
    Logger.log('Setup complete for sheet: ' + sheetName);
  }
  
  // Set up data validation on Products sheet
  setupProductValidation(ss);
  
  // Set up number formatting
  setupFormatting(ss);
  
  // Remove default "Sheet1" if it exists and is empty
  var defaultSheet = ss.getSheetByName('Sheet1');
  if (defaultSheet && defaultSheet.getLastRow() <= 1) {
    ss.deleteSheet(defaultSheet);
  }
  
  SpreadsheetApp.getUi().alert('✅ All sheets created and configured successfully!\n\nSheets created:\n• Products\n• Bills\n• GST_Bills\n• Stock_Log');
}

// ============================================================
// DATA VALIDATION
// ============================================================
function setupProductValidation(ss) {
  var productSheet = ss.getSheetByName('Products');
  if (!productSheet) return;
  
  // GST % validation (column E = 5th column)
  var gstRule = SpreadsheetApp.newDataValidation()
    .requireValueInList(['0', '5', '12', '18', '28'], true)
    .setAllowInvalid(false)
    .setHelpText('Select valid GST rate: 0, 5, 12, 18, or 28%')
    .build();
  productSheet.getRange('E2:E1000').setDataValidation(gstRule);
  
  // Unit validation (column H)
  var unitRule = SpreadsheetApp.newDataValidation()
    .requireValueInList(['Piece','Kg','Gram','Litre','ML','Box','Dozen','Pack','Set','Metre'], true)
    .setAllowInvalid(true)
    .setHelpText('Select unit of measure')
    .build();
  productSheet.getRange('H2:H1000').setDataValidation(unitRule);
  
  // Bills payment mode (column L)
  var billsSheet = ss.getSheetByName('Bills');
  if (billsSheet) {
    var payRule = SpreadsheetApp.newDataValidation()
      .requireValueInList(['Cash','Card','UPI','Credit'], true)
      .setAllowInvalid(false)
      .build();
    billsSheet.getRange('L2:L5000').setDataValidation(payRule);
  }
  
  Logger.log('Data validation applied');
}

// ============================================================
// NUMBER FORMATTING
// ============================================================
function setupFormatting(ss) {
  // Products — Price column (D)
  var productSheet = ss.getSheetByName('Products');
  if (productSheet) {
    productSheet.getRange('D2:D1000').setNumberFormat('₹#,##0.00');
    productSheet.getRange('E2:E1000').setNumberFormat('0"%"');
    productSheet.getRange('G2:G1000').setNumberFormat('#,##0');
  }
  
  // Bills — currency columns
  var billsSheet = ss.getSheetByName('Bills');
  if (billsSheet) {
    billsSheet.getRange('F2:K5000').setNumberFormat('₹#,##0.00');
  }
  
  // GST_Bills — currency columns
  var gstSheet = ss.getSheetByName('GST_Bills');
  if (gstSheet) {
    gstSheet.getRange('H2:P5000').setNumberFormat('₹#,##0.00');
  }
  
  Logger.log('Number formatting applied');
}

// ============================================================
// CONDITIONAL FORMATTING — Color-code stock levels
// ============================================================
function applyStockFormatting() {
  var ss    = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('Products');
  if (!sheet) { Logger.log('Products sheet not found'); return; }
  
  var range = sheet.getRange('G2:G1000'); // Quantity column
  
  // Clear existing rules
  sheet.clearConditionalFormatRules();
  var rules = [];
  
  // Out of stock (Qty = 0) — Red background
  var outRule = SpreadsheetApp.newConditionalFormatRule()
    .whenNumberEqualTo(0)
    .setBackground('#fef2f2')
    .setFontColor('#dc2626')
    .setBold(true)
    .setRanges([range])
    .build();
  rules.push(outRule);
  
  // Low stock (Qty 1–10) — Yellow background
  var lowRule = SpreadsheetApp.newConditionalFormatRule()
    .whenNumberBetween(1, 10)
    .setBackground('#fffbeb')
    .setFontColor('#d97706')
    .setRanges([range])
    .build();
  rules.push(lowRule);
  
  // Good stock (Qty > 10) — Green background
  var goodRule = SpreadsheetApp.newConditionalFormatRule()
    .whenNumberGreaterThan(10)
    .setBackground('#f0fdf4')
    .setFontColor('#16a34a')
    .setRanges([range])
    .build();
  rules.push(goodRule);
  
  sheet.setConditionalFormatRules(rules);
  SpreadsheetApp.getUi().alert('✅ Stock color formatting applied to Products sheet!');
}

// ============================================================
// DAILY SALES SUMMARY — Creates a summary report
// ============================================================
function generateDailySummary() {
  var ss         = SpreadsheetApp.getActiveSpreadsheet();
  var billsSheet = ss.getSheetByName('Bills');
  if (!billsSheet) { Logger.log('Bills sheet not found'); return; }
  
  var today     = Utilities.formatDate(new Date(), 'Asia/Kolkata', 'yyyy-MM-dd');
  var data      = billsSheet.getDataRange().getValues();
  var headers   = data[0];
  var dateIdx   = headers.indexOf('Date');
  var amtIdx    = headers.indexOf('Final_Amount');
  var modeIdx   = headers.indexOf('Payment_Mode');
  
  var totalSales = 0;
  var billCount  = 0;
  var byMode     = {};
  
  for (var i = 1; i < data.length; i++) {
    var row = data[i];
    if (String(row[dateIdx]).trim() === today) {
      var amt = parseFloat(row[amtIdx]) || 0;
      totalSales += amt;
      billCount++;
      var mode = String(row[modeIdx]);
      byMode[mode] = (byMode[mode] || 0) + amt;
    }
  }
  
  var report = '📊 DAILY SALES SUMMARY\n';
  report += '═══════════════════════\n';
  report += 'Date: ' + today + '\n';
  report += 'Total Bills: ' + billCount + '\n';
  report += 'Total Sales: ₹' + totalSales.toFixed(2) + '\n\n';
  report += 'BY PAYMENT MODE:\n';
  for (var m in byMode) {
    report += '  ' + m + ': ₹' + byMode[m].toFixed(2) + '\n';
  }
  
  SpreadsheetApp.getUi().alert(report);
}

// ============================================================
// LOW STOCK ALERT — Find products running low
// ============================================================
function checkLowStockAlert() {
  var ss           = SpreadsheetApp.getActiveSpreadsheet();
  var productSheet = ss.getSheetByName('Products');
  if (!productSheet) return;
  
  var data        = productSheet.getDataRange().getValues();
  var headers     = data[0];
  var nameIdx     = headers.indexOf('Name');
  var qtyIdx      = headers.indexOf('Quantity');
  var LOW_THRESH  = 10;
  
  var outItems = [];
  var lowItems = [];
  
  for (var i = 1; i < data.length; i++) {
    var row = data[i];
    if (!row[nameIdx]) continue;
    var qty = parseInt(row[qtyIdx]) || 0;
    if (qty <= 0)           outItems.push(row[nameIdx] + ' (OUT OF STOCK)');
    else if (qty <= LOW_THRESH) lowItems.push(row[nameIdx] + ' (' + qty + ' left)');
  }
  
  if (!outItems.length && !lowItems.length) {
    SpreadsheetApp.getUi().alert('✅ All products are well stocked!');
    return;
  }
  
  var msg = '';
  if (outItems.length) msg += '🔴 OUT OF STOCK:\n' + outItems.join('\n') + '\n\n';
  if (lowItems.length) msg += '🟡 LOW STOCK:\n' + lowItems.join('\n');
  SpreadsheetApp.getUi().alert('⚠️ STOCK ALERT\n\n' + msg);
}

// MENU — Adds a custom menu to the spreadsheet

function onOpen() {
  var ui = SpreadsheetApp.getUi();

    .addItem('⚙️ Setup All Sheets', 'setupSheets')
    .addItem('🎨 Apply Stock Colors', 'applyStockFormatting')
    .addSeparator()
    .addItem('📊 Today\'s Sales Summary', 'generateDailySummary')
    .addItem('⚠️ Check Low Stock', 'checkLowStockAlert')
    .addSeparator()
    .addItem('🗑️ Clear All Bills Data', 'clearBillsData')
    .addToUi();
}

// ============================================================
// UTILITY — Clear Bills (use with caution!)
// ============================================================
function clearBillsData() {
  var ui = SpreadsheetApp.getUi();
  var response = ui.alert(
    '⚠️ WARNING',
    'This will delete ALL bill records. This cannot be undone!\n\nAre you sure?',
    ui.ButtonSet.YES_NO
  );
  
  if (response === ui.Button.YES) {
    var ss         = SpreadsheetApp.getActiveSpreadsheet();
    var billsSheet = ss.getSheetByName('Bills');
    if (billsSheet && billsSheet.getLastRow() > 1) {
      billsSheet.deleteRows(2, billsSheet.getLastRow() - 1);
      ui.alert('✅ Bills data cleared.');
    }
  }
}
```

## Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| `Failed to get access token` | Wrong private key or client email | Re-check credentials.json values in .env |
| `Requested entity was not found` | Wrong Spreadsheet ID | Copy ID again from the URL |
| `The caller does not have permission` | Sheet not shared with service account | Share spreadsheet with the service account email |
| `Google Sheets API not enabled` | API disabled | Enable it in Cloud Console → Library |
| `SSL certificate error` | cURL SSL issue on WAMP | Download cacert.pem and set in php.ini |

### Fix SSL on WAMP (php.ini):
```ini
curl.cainfo = "C:/wamp64/bin/php/php5.6.40/extras/ssl/cacert.pem"
openssl.cafile = "C:/wamp64/bin/php/php5.6.40/extras/ssl/cacert.pem"
```
Download cacert.pem from: https://curl.se/ca/cacert.pem
